⚡ ORIGIN STORY

About the Site

The idea for UniMinds Arena came from an A-Level Physics lesson. My teacher played a game with the class (called "University Challenge") where our teams or "Universities" would answer questions being read aloud from a subject knowledge organiser. Unfortunately for our poor teacher, everyone in the class always argued about who said their team name first. In a throwaway comment our teacher mentioned the idea of a website-based game being made to solve this problem; and the website was born.

— Harry Muscroft
🔒 PRIVACY POLICY

Privacy & Data

UniMinds Arena is designed to be simple, private and safe to use in a school environment. Here is how data is handled:

  • Teacher email addresses are used solely to send a one-time verification code (OTP) and to maintain a login session. Email addresses are stored only in your browser's localStorage and are never sent to, or stored on, any external server beyond the email delivery service (EmailJS) used to dispatch the verification code.
  • Verification codes expire after 10 minutes and are generated entirely in the browser — they are never logged or persisted on the server.
  • Student names are entered voluntarily when joining a game. They are held in server memory only for the duration of the active game session and are permanently deleted when the session ends or the server restarts.
  • Lobby and game data (room codes, question sets, scores, team names) exists only in server memory during an active session. No game data is written to disk or stored in a database. When the host ends the session or the server restarts, all data is gone.
  • Room codes are randomly generated six-letter strings. They are not tied to any personal information and expire as soon as the session is closed.
  • No cookies are set by this application beyond standard session storage used to track which game a student is currently in.
  • No third-party analytics, advertising trackers, or data brokers are used anywhere on this site.

If you have any questions about how your data is handled, or would like something removed, please get in touch using the contact details below.
